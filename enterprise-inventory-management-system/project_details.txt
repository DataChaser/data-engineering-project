# Enterprise Inventory Management System

A SQL-based inventory system designed to simulate a realistic enterprise setup. This project showcases schema design, normalization, data handling, stored procedures, and triggers.

## What's Included

- create_tables.sql – All database tables  
- insert_data.sql – Realistic dataset (these are dummy data just for portfolio use)  
- stored_procedures.sql – Two simple procedures for placing and receiving orders  
- triggers.sql – Triggers for logging inventory changes and flagging low stock  
- sample_queries.sql – Five useful SQL queries  
- er_diagram.png – Visual schema overview  

## Schema Overview

- users
- stores
- suppliers
- products
- inventory
- orders
- order_items

Normalized with foreign keys and constraints.

## Getting Started

1. Run create_tables.sql  
2. Run insert_data.sql  
3. Run stored_procedures.sql and triggers.sql  
4. Use sample_queries.sql for exploration  
5. View er_diagram.png for table relationships
